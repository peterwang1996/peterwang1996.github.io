hello, world

发表于 2016-07-28 | 更新于 2016-07-31

感谢腾讯爸爸的鼎力支持，让我们这样的穷学生能够在每个月只交一块钱保护费，就能在网上有个能写一点文字，并且能偶尔做点小实验的地方。在这里特别鸣谢腾讯爸爸，代表全人类感谢你。（现在换成gh-pages了）

话说这个 CentOS + nginx + node.js + MariaDB + ghost.js 的环境，因为之前没有这方面经验（如果不算在长大在线搭过好几遍的 LAMP 环境的话），而且因为没什么人玩这个，所以网上教程相当少，而且几乎每个都有坑（因为 node.js 和 ghost.js 的变化实在是太快），所以搭起来这个环境的过程其实还蛮艰辛的。要不是腾讯云的超快无限次重装系统功能，恐怕这玩意儿我折腾到八月份都不一定算完。所以，在这个博客系统安定下来以后，我可能会写一点东西来记录这些过程，供以后踩坑的同学做参考。

那么，既然坑都开了，总得做点什么才对得起它吧。所以现在在这里把希望用这个博客系统和这个服务器做的一些事情在这里做个备忘。
